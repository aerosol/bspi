# bspi

Automatic desktop renamer for bspwm.

Recognizes clients and renames desktop clients to their pre-configured
FontAwesome icons.

## Prerequisites:

bspwm (duh), xprop, python3

## Installation:

Run `bspi_listen &` in your `bspwmrc`.
